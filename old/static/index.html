About the NIAGADS API

NIAGADS has developed several databases and tools in support of Alzheimer's disease (AD) research. Our application programming interface (API) provides programmatic accesses to these resources, allowing users to integrate our data and annotations into their own analysis pipelines, facilitating investigations at chromosome- and genome-wide scales.

The NIAGADS API uses HTTP requests to access and disseminate data from unrestricted, public NIAGADS knowledgebases. It has predictable resource- and genomic-feature oriented URLs and returns JSON-encoded responses, associated with standard HTTP response codes.

NIAGADS Resources with API Access

The NIAGADS API provides access to all public NIAGADS databases. These incude the following:

genomicsDB

The NIAGADS Alzheimer's Genomics Database

is disease-centric, interactive knowledgebase for Alzheimer's disease (AD) genetics. It provides a platform for data sharing, discovery, and analysis by making available AD-relevant GWAS summary statistics datasets from the NIAGADS repository for browsing and mining. Also available are detailed reports that compile genetic evidence for AD-risk with functional annotations in the context of gene and variants.

https://www.niagads.org/genomics

Use the NIAGADS API to:

  • Find the top risk-associated variants in AD-relevant GWAS summary statistics datasets.
  • Get AD-relevant genetic and functional annotations in the context of a gene or variant of interest.
  • Get Alzheimer's Disease Sequencing Project (ADSP) variant QC results and annotations.
  • Fetch annotations for a single gene or variant or do efficient bulk lookups (feature lists or within a genomic region).

FILER

FILER Functional Genomics Repository

is a disease-agnostic functional genomics database developed by NIAGADS with the goal of establishing a comprehensive harmonized, extensible, indexed, searchable human functional genomics data collection across >20 data primary data sources. FILER currently contains >75k searchable functional genomics tracks. All tracks are annotated with extensive metadata that captures information about the provenance, biosamples, and experimental design.

https://tf.lisanwanglab.org/FILER/

Use the NIAGADS API to:

  • Query FILER metadata to find tracks relevant to your analysis.
  • Search identified tracks singly or in bulk to find hits within a genomic span of interest.
